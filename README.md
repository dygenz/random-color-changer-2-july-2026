# Random Color Generator
A simple random color generator which pick color randomaly and change the background color of the page

## Features
- Generate Random colors
- Press the space to generate a new color
- Display the current color in the the middle in a box
- Keep last 5 colors in the history box in right bottom of the page

## Technology
- Html
- Css
- JavaScript

## How to run 
- Download the project
- Open index.html in your browser
